Introduction

TAIA is meant to make life easier for teachers. At it's core, TAIA helps educators make, grade and review assessments like homework and tests. The AI in TAIA referes to algorithms capable of reading and grading tests without the use of an expensive scantron. TAIA also uses machine learning to Help educators review assessments, learning about their students and about how to make assessments more effective and fair.

Download and Install

It is installable on windows 8.1 and higher as well as Max OS X and Ubuntu. The Software can be downloaded from here PyInstaller is used to build a single file executable that is easy to use for those that don't want to build from source.

Version 0.1.0 Scheduled Features

  • Easy Download and Launch: Download File, click it To Run
  • Persistant Mempry: TAIA remembers where you keep your files
  • Graphical User Interface for easy navigation and usage
  • Assessment Generator: GUI based assessment builder for making multiple choice and free response questions
  • Assessment PDF Builder: Converts an assessment into pdf form that can be printed and given to student to take. This form is designed to improve the accuracy of TAIA's Computer Vision Grading
  • Computer Vison Grading: Scan all of your tests as a single big using your local Copy Machine. Then let TAIA do the grading For you using it's Computer Vision
  • Review Graded Assessments using interactive data mining tools

Looking to The Future

TAIA is both free software for teachers as well as vehicle for research in AI.
  • Improved human handwriting recognition
  • More flexible / intuitive GUI
  • Word Content Analysis: Look for patterns in semantic content of questions and answers
  • Text Book Incorporation: Use knowledge from text books to improve automation of grading and topic modeling. Recommendations for review concepts for students
